About

Documentation of the Development Process


The web site is done for fictional uncle, who has requested help in developing a website for his small business. The website structure I chose to use is the hierarchies. Hierarchical structure planned around a home page. The home page use as the core navigation location. All the other pages have a link through the home page and in reverse as well. Other pages organize as a tree structure or multi-tiered. Home page navigation bar with all other sub-categories include Tea, Teaware, about us, FAQ and contact us as sub-categories. This structure is easier to navigate for the users so they could find the products easier.
